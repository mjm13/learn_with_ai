# 如何使用英语私教系统

## 🚀 快速开始

### 步骤1: 启动私教
1. 打开你的AI助手(ChatGPT、Claude等)
2. 复制 `english_tutor_prompt.md` 中的完整内容
3. 粘贴发送给AI
4. AI将以英语私教身份开始与你对话

### 步骤2: 完成初始评估
- 私教会给你进行3项测试:
  - 词汇量测试(10题)
  - 阅读理解(1段)
  - 语法检测(5题)
- 如实回答,不要查字典
- 评估后会得到你的水平定级

### 步骤3: 开始学习
- 私教会根据你的水平制定学习计划
- 每次学习20-30分钟
- 建议每天或每周固定时间学习

### 步骤4: 跟踪进度
- 每次学习结束后,私教会总结本次内容
- 手动或让AI帮你更新 `progress/learning_log.md`
- 定期查看进度,保持动力

## 💡 使用技巧

### 1. 保持对话连续性
- 使用同一个AI对话串进行学习
- 如果切换了对话,需要:
  - 重新发送私教prompt
  - 告知当前学习进度
  - 提供 `learning_log.md` 的内容

### 2. 选择合适的学习模式
- **时间充足**: 选择"阅读训练"(30-45分钟)
- **时间有限**: 选择"词汇学习"(15-20分钟)
- **想巩固**: 选择"复习测验"(20分钟)

### 3. 及时更新进度
- 每次学习后立即更新日志
- 可以让AI帮你生成更新内容
- 复制粘贴到 `learning_log.md`

### 4. 定期复习
- 每周末复习本周词汇
- 每月做一次综合测验
- 查看进度日志,强化记忆薄弱的部分

## ⚙️ 自定义设置

你可以根据自己的需要调整私教的行为:

### 修改学习节奏
在与私教对话时告诉它:
- "每次只学15个新词,不要太多"
- "我想加快进度,每次学40个词"
- "我每周只能学习2次"

### 聚焦特定领域
- "我主要做前端开发,多学点React相关的"
- "我需要读数据库文档,多教点SQL术语"
- "我最近在学Python,给我Python相关的材料"

### 调整难度
- "这个太难了,给我简单一点的"
- "我觉得太简单,可以提升难度吗"

## 🎯 学习建议

### 最佳实践
1. ✅ **固定时间学习**: 每天同一时间,养成习惯
2. ✅ **小步快跑**: 每次20-30分钟,不要贪多
3. ✅ **及时复习**: 当天学的当天复习一次
4. ✅ **实战应用**: 学完立即找真实文档试读
5. ✅ **记录进步**: 定期查看学习日志,看到成长

### 常见问题

**Q: 学了就忘怎么办?**
A: 正常现象。使用间隔复习,第1天、第3天、第7天、第15天各复习一次。

**Q: 有些词总是记不住?**
A: 告诉私教哪些词记不住,它会:
- 多次以不同方式出现
- 提供更多例句
- 在测验中重点考察

**Q: 进度很慢,有点沮丧?**
A: 查看累计数据,你可能已经学了几百个词!语言学习需要时间,坚持最重要。

**Q: 如何知道自己进步了?**
A: 每月做一次评估测试,或者找一篇之前读不懂的文档,现在再试试看!
